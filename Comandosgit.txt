Inicializa um repositório local
- git init

Criar uma branch local
- git branch -M main

Conecta um repositório local a um remoto
- git remote add origin https://github.com/LucasPopowicz/react---native---senac.git

Adiciona arquivos ao commit
- git add <ARQUIVOS>

Efetua o commit no repositório local
- git commit -m "Primeiro commit"

Transfere os arquivos comitados localmente para o remoto
- git push -u origin main

//Verificar qual repositório remoto está conectado
- git remote -v
